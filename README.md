# music-app
ReactJS & Redux Music player. Feel free to use it, change, or contribute.

### To run the project:
``` 
$ pnpm i
$ pnpm run dev
```

### Works currently:
1) Songs repeat/shuffle
2) localStorage cache (last song, repeat etc)
3) AudioSession api features (now-playing, hotkeys next/prev)
4) Cover expand
